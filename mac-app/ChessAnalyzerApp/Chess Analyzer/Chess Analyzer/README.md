# Chess Analyzer

A powerful chess analysis tool for macOS.

## Features

- ♟️ Interactive chess board with legal move validation
- 📊 Real-time position evaluation with Stockfish
- 📝 Move notation and game tree support
- 🗂️ PGN import/export
- 💾 Local game database with search
- 📈 Evaluation bar (chess meter)
- 🌳 Variation/branch support

## Quick Start

1. Open the project in Xcode
2. Build and run (⌘R)
3. Start analyzing chess games!
